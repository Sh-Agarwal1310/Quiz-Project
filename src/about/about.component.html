About Us


Quiz time is a single page angular application which conducts online quiz and displays the results.

The web page has used Angular Modules, Angular Components, Angular Templates, Angular routing, Angular Material, Angular DIrectives and further more functionalities of Angular.

It offers three different quizes for java, python and java script.

It allows the user to choose the correct option and submit the test. User can review the questions that are left answered or unanswered.

After the submittion, final score would be displayed.
